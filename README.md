# news-card-images

Daily news card image hosting for @allround_news

## 카드 발행 일정

### 평일 (한국 + 미국 양쪽 모두 개장일)

| 시간 | 카드 | 주요 내용 |
|------|------|-----------|
| 07:00 | 미장 마감 | 나스닥/S&P/다우, 주요 종목, 빅이슈 |
| 16:00 | 한국장 마감 | 코스피/코스닥, 수급, 주요 종목 |

### 휴장일 (한국 공휴일 OR 미국 공휴일 + 주말)

| 카드 | 주요 내용 |
|------|-----------|
| 인사이트 | 주간 핫 섹터 정리 + 투자 인사이트 |

## 카드 파일 네이밍

```
card_YYYYMMDD_us.png   # 미장 카드
card_YYYYMMDD_kr.png   # 한국장 카드
card_YYYYMMDD.png      # 휴일 인사이트 카드
```
