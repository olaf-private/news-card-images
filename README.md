# news-card-images

Daily news card image hosting for @allround_news

## 카드 발행 일정

각 슬롯은 독립적으로 판단합니다.

### 07시 슬롯 (미국장 기준)

| 미국장 | 발행 카드 | 주요 내용 |
|--------|-----------|-----------|
| 개장 | 미장 마감 카드 | 나스닥/S&P/다우, 주요 종목, 빅이슈 |
| 휴장 | 미장 인사이트 카드 | 미국 시장/경제 관련 핫 섹터 + 투자 인사이트 |

### 16시 슬롯 (한국장 기준)

| 한국장 | 발행 카드 | 주요 내용 |
|--------|-----------|-----------|
| 개장 | 한국장 마감 카드 | 코스피/코스닥, 수급, 주요 종목 |
| 휴장 | 한국장 인사이트 카드 | 한국 시장/경제 관련 핫 섹터 + 투자 인사이트 |

### 케이스별 요약

| 미국 | 한국 | 07시 | 16시 |
|------|------|------|------|
| 개장 | 개장 | 미장 카드 | 한국장 카드 |
| 휴장 | 개장 | 미장 인사이트 카드 | 한국장 카드 |
| 개장 | 휴장 | 미장 카드 | 한국장 인사이트 카드 |
| 휴장 | 휴장 | 미장 인사이트 카드 | 한국장 인사이트 카드 |

## 카드 파일 네이밍

```
card_YYYYMMDD_us.png         # 미장 마감 카드
card_YYYYMMDD_kr.png         # 한국장 마감 카드
card_YYYYMMDD_us_insight.png # 미장 인사이트 카드
card_YYYYMMDD_kr_insight.png # 한국장 인사이트 카드
```
